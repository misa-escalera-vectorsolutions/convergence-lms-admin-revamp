# Legacy To Refreshed Mapping

| Legacy Element | Refreshed Implementation | Functional Change |
| --- | --- | --- |
| Top application nav | `vwc-topnav` with the same product-level links | None |
| Left Administration rail | `vwc-sidenav` with the same section labels | None |
| Environment and dashboard context bar | Location chip + `vwc-bread-crumb-nav` + Help button | None |
| Admin Quick Access container | `vwc-card` panel | None |
| Six blue admin tiles | Six refreshed quick-access tiles generated from data | None |
| Help action | `vaadin-button` plus `vwc-topnav` help affordance | None |
| Footer metadata | Updated footer text inside the quick-access panel | None |

## Key Constraint

This prototype intentionally avoids introducing a new dashboard model, alternate workflow hierarchy, or additional operational modules. The goal is a fresh look with Vector styling while keeping the Administration page functionally familiar.
